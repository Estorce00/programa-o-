VOCE SABE O QUE É HTML?

• HTML •

HyperText Markup Language (em português: Linguagem de Marcação de Hipertexto).
“HyperText” → significa texto que contém links (hiperlinks) para outras páginas ou recursos.
“Markup Language” → significa linguagem de marcação, ou seja, usada para marcar (estruturar) os elementos de um documento.
